# Smart PG Management System

## 📌 Project Overview

The Smart PG Management System is a console-based Python application developed to simplify and automate hostel and paying guest management operations. The system helps administrators manage residents, room allocations, rent payments, visitor records, and room status efficiently.

This project was developed using Python and demonstrates the practical implementation of programming concepts such as functions, lists, dictionaries, loops, conditional statements, exception handling, and input validation.

---

## 🚀 Features

- Secure Admin Login System
- Add Resident Details
- Mobile Number Validation
- Room Sharing Management (2, 3, and 4 Sharing)
- Automatic Rent Calculation
- Room Allocation System
- Rent Payment Tracking
- Visitor Entry Management
- View Resident Records
- Search Resident Information
- Room Status Monitoring
- Hostel Report Generation
- Occupied and Vacant Room Checking

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open Command Prompt or Terminal.
4. Navigate to the project folder.
5. Run the following command:

```bash
python pg.py
```

6. Login using the admin credentials.
7. Use the menu options to manage hostel operations.
